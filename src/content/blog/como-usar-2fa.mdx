# Como Usar Autenticação de Dois Fatores (2FA)

A **autenticação de dois fatores (2FA)** é uma das medidas de segurança mais eficazes que você pode adotar hoje. Mesmo que sua senha seja comprometida, o 2FA impede que invasores acessem sua conta.

## O que é 2FA?

O 2FA adiciona uma segunda camada de verificação além da senha. Quando você faz login, precisa fornecer:

1. **Algo que você sabe** — sua senha
2. **Algo que você tem** — código do celular, chave física, etc.

## Tipos de 2FA

### Aplicativos Autenticadores (Mais Seguro)
- Google Authenticator
- Authy
- Microsoft Authenticator

Geram códigos TOTP de 6 dígitos que expiram a cada 30 segundos.

### SMS (Conveniente, mas Menos Seguro)
Vulnerável a ataques de SIM Swap. Use apenas se não houver outra opção.

### Chaves de Segurança Física (Máxima Segurança)
- YubiKey
- Titan Security Key

Ideal para contas críticas como email principal e banco.

## Como Ativar em Cada Serviço

### Google
Acesse: Conta Google → Segurança → Verificação em duas etapas

### Instagram/Facebook
Configurações → Segurança → Autenticação de dois fatores

### Bancos Brasileiros
A maioria dos bancos digitais (Nubank, Itaú, Bradesco) já exige 2FA nativo.

## Senhas Fortes + 2FA = Proteção Real

Combine o 2FA com senhas fortes geradas pela nossa [ferramenta gratuita](/tools/senha). Uma senha de 16+ caracteres aleatórios + 2FA torna uma conta virtualmente inviolável.
