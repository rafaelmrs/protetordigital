Esse link é seguro para clicar?

Recebeu um link suspeito pelo WhatsApp, e-mail ou SMS? Verifique antes de abrir — golpistas usam links falsos para roubar seus dados.

Sinais de link perigoso
  • URL muito longa e cheia de letras aleatórias
  • Nome do banco ou loja escrito errado (bb.com → bbb.com)
  • Promessas urgentes: "seu CPF será bloqueado"
  • Links encurtados enviados por desconhecidos
  • Começa com um número de IP (ex: 192.168.1.1)
Como funciona a verificação
Este verificador usa o Google Safe Browsing, o mesmo sistema que protege o Chrome e o Gmail, para identificar sites de phishing, malware e golpes conhecidos. O link é verificado, não armazenado.
